# 🏥 Healthcare Analytics Dashboard

An interactive Healthcare Analytics Dashboard built using **Power BI, SQL Server, DAX, and Excel** to provide actionable insights into hospital performance, patient demographics, and financial operations.

---

## 📌 Project Overview

This dashboard helps healthcare administrators and decision-makers monitor hospital performance through interactive visualizations and key performance indicators.

The dashboard is divided into four analytical pages:

- 📊 Executive Dashboard
- 🏥 Hospital Performance
- 👥 Patient Analytics
- 💰 Financial Analytics
